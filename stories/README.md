# Stories

This directory holds root-level Storybook stories that are broader than a single component file.

## Use This For

- design system overview pages
- foundations demos
- shared pattern demos
- cross-component examples

## Do Not Use This For

- domain logic
- infrastructure examples
- route wiring
- one-off feature components that belong next to their source file

## Structure

- `overview/` for high-level design-system or product UI overview stories
- `foundations/` for tokens, typography, spacing, and motion demos
- `patterns/` for shared UX pattern stories
