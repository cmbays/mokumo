# Screen Print Pro

Production management software for **4Ink**, a screen-printing shop. Manages the full garment lifecycle:

**Quote → Artwork Approval → Screen Room → Production → Shipping**

## What This Is

Screen Print Pro replaces spreadsheets, text messages, and memory with a unified production dashboard. The shop owner gets instant clarity on job status, blocked items, and next actions.

## Who It's For

**Primary user**: Shop owner/operator running a small screen-printing business (1-5 employees), managing 5-20 active jobs at any time. Values speed and clarity over feature count.

## Service Types

The system supports three decoration methods, each with its own quoting flow and production pipeline:

| Service | Description | Production Steps |
|---------|-------------|-----------------|
| **Screen Printing** | Traditional ink-on-garment via burned screens | Art → Film → Burn → Register → Press → QC |
| **DTF (Direct to Film)** | Transfer film printed and applied to garments | Art → Gang Sheet → Print → Press → QC |
| **DTF Press** | Shop presses customer-supplied DTF transfers | Receive → Press → QC |

## Current Phase

**Phase 2** — Backend foundation + vertical builds. Phase 1 (high-fidelity mockups) is complete with all 7 verticals demo-ready.

## Tech Stack (Summary)

| Layer | Technology |
|-------|-----------|
| Framework | Next.js 16 (App Router, TypeScript, Turbopack) |
| Styling | Tailwind CSS v4 + shadcn/ui |
| Database | Supabase PostgreSQL + Drizzle ORM |
| Auth | Supabase Auth (email/password) |
| Cache | Upstash Redis |
| Analytics | dbt-core (medallion pipeline) |
| Deployment | Vercel (preview on main, production branch) |
| Supplier API | S&S Activewear REST V2 |
