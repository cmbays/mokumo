# M9: Beta Launch

> **Status**: Horizon
> **Exit signal**: Multiple shops using Mokumo daily. NPS > 40. V1 GA scope locked.

Real shops. Real feedback. The flywheel starts.

## What Ships

| Component                 | Key Deliverables                                                                        |
| ------------------------- | --------------------------------------------------------------------------------------- |
| 2–3 beta shops live       | Shops using Mokumo as primary system for ≥ 1 week without falling back                  |
| Feedback loop operational | User issues → research → synthesis → prioritized fixes → shipped. Closes within 2 weeks |
| Beta feedback channels    | In-app feedback widget. Weekly async check-in with each beta shop                       |
| NPS baseline              | NPS survey after 2 weeks. Target: > 40                                                  |
| V1 GA scope defined       | What needs to ship before charging money? Betting session from beta feedback            |

## Key Decisions (Directional)

- Beta shops use Mokumo as their primary tool — not a side experiment
- Feedback loop closes within 2 weeks (issue → fix → shipped)
- V1 GA scope decided from beta learnings, not pre-planned

## Depends On

- M8 beta readiness (operational maturity)
- All critical user journeys working reliably
